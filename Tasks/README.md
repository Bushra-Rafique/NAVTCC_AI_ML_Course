# NAVTCC — Artificial Intelligence, Machine Learning & Deep Learning

> A complete hands-on task repository covering Python fundamentals, data science libraries, and machine learning — developed as part of the **NAVTCC AI/ML/Deep Learning Course**.

---

## Overview

This repository contains **56 Jupyter Notebooks** organized progressively from Python basics to advanced machine learning and deep learning concepts. Each notebook corresponds to a course task and is self-contained with explanations, code, and outputs.
